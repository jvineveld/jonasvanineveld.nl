Plain HTML templates

Minimale code en veel resultaat. Bij plain HTML templates werk ik enkel een statisch document uit bestaande uit HTML, CSS (sass of less) en JavaScript.

Deze templates zijn niet helemaal functioneel, maar dienen als basis voor backend developers om hun code op aan te sluiten.

Ik schrijf HTML5 valide code, en zorg dat alles cross-browser lekker werkt.

Grafische visuals

Door websites heen, op campagne pagina's, of in banners wil je soms wat meer dan alleen een statische mooi design.

Goed gebruik transitities en animaties, kan een UX duidelijker en moderner maken.

Ik schrijf simpele animaties met behulp van CSS transities en animaties.

Uitgebreide animaties maak ik met behulp van de javascript GSAP bibliotheek.

Laadtijd & rendering optimalisatie

De laadtijd van je website is extreem belangrijk voor een goede conversie. Maar behalve de laadtijd, is de gebruikerservaring na het laden ook belangrijk.

Scrolt alles lekker vloeiend? geen haperingen? Zijn de animaties soepel, en hoe is dat voor een oude android telefoon?

Een website kan nog zo mooi en slim gemaakt zijn, maar als het niet soepel werkt is de ervaring voor de gebruiker slecht.

In het verleden heb ik verschillende grote website's ge-optimaliseerd.
